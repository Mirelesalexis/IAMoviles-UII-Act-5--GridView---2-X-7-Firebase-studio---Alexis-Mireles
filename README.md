Prompt para Generar el GridView de Dominos
"Actúa como un desarrollador experto en Flutter. Necesito crear una aplicación de catálogo para 'Dominos Pizza' usando un GridView.builder.

Requerimientos técnicos:

Estructura del Grid: Debe tener exactamente 2 columnas.

Contenido: Una lista de 14 elementos (7 filas) que incluya Pizzas, Bebidas y Complementos.

Diseño de Tarjeta (Card): Cada celda del Grid debe ser una tarjeta con bordes redondeados que contenga:

Una imagen en la parte superior (usa Image.network con links de placeholder por ahora).

Un área de texto debajo de la imagen con 3 niveles:

Título: Nombre del producto en negrita.

Subtítulo: Descripción corta en color gris.

Rating: Una fila de 5 iconos de estrellas (Icons.star) en color amarillo/ámbar.

Estilo: Usa los colores corporativos de Dominos (Azul: 0xFF006491 y Rojo: 0xFFE31837).

Código Limpio: Separa los datos en una lista de Mapas (List<Map<String, String>>) para que sea fácil de mantener.

Entrega el código completo, listo para copiar y pegar en el archivo main.dart."
